Git

  • It may seem obvious, but you should always start with "git" for any git commands in terminal
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Once you have added the content you want, there are several steps you must take to get your code merged to main branch
      • Stage your branch by "git add -A"
      Commit your changes by "git commit -m message notes here"
      Confirm your current main branch is up to date "git pull origin branch"
      Push your changes " git push origin "
      Go to Github to merge your branch into main branch
      Create a pull request; You will see your branch compared with the main one.
      Merge pull request
  • Update local repository with new merged in feature
  • git pull origin main
      • You can find all local branches by "git branch"
